== Tradable Analysis Toolset

Tradable Analysis Toolset (TAT) is a Ruby-on-Rails-based web application
that provides a set of features that support the use of
{technical analysis}[http://en.wikipedia.org/wiki/Technical_analysis] for
making trading decisions with respect to stocks and derivatives.  TAT uses
the {Market Analysis System}[https://github.com/jjttcc/market-analysis-system]
as its underlying engine for the required technical analysis calculations
and related functionality.
